# CTG-METIERS - GTA-V

Ce script est à but éducatif.

- Possibilité de faire le job bucheron/mineur et autres seulement en se déplacant vers les checkpoints
- Possibilité de supprimer des éléments de son inventaire en un click
- Possibilité de courir/conduire automatiquement
